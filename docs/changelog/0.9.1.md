# v0.9.1 - GUI Fixes & Command Improvements

## Overview
Patch release addressing PR review feedback with permission enforcement, command improvements, and test coverage.

## Fixes

### Permission Enforcement
- **Hot Chunks Teleport**: Added permission check for teleporting to hot chunks
  - Requires `smpstats.gui.health.manage` or `smpstats.admin` permission
  - Players without permission now receive an error message and sound

### Command Improvements
- **Tab Completion**: Added tab completion for `/stats` and `/smpstats` commands
  - Previously only `/sstats` had tab completion registered

### Code Quality
- **ServerHealthService**: Unknown player fallback now uses full UUID instead of truncated 8-character substring
- **Test Coverage**: Added comprehensive tests for teleport permission scenarios

## Technical Details

### Files Changed
- `SMPStats.java`: Register TabCompleter for all stat commands
- `HotChunksGui.java`: Add permission check in `performTeleport()`
- `ServerHealthService.java`: Use full UUID in unknown player fallback
- `SStatsCommandTest.java`: Implement proper GUI command test with Bukkit mocking
- `HotChunksGuiTest.java`: Add teleport permission denial/approval tests

### Test Coverage
- Overall coverage: 91%
- All 342 tests passing
