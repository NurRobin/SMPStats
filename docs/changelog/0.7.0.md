# 0.7.0 - Analytics Layer Enhancements

### Features
- **Heatmap Grid Sizes**: Added support for configurable grid sizes (8, 16, 32, 64) in heatmap generation.
- **New Heatmap Layers**:
  - `DAMAGE`: Tracks locations where players take damage.
  - `MINING_<ORE>`: Tracks mining locations for specific ores (e.g., `MINING_DIAMOND_ORE`).
- **Dynamic Aggregation**: Heatmaps are now aggregated on-demand based on query parameters.
- **API Update**: Added `grid` parameter to `/heatmap` endpoint.

### Improvements
- Optimized heatmap generation logic.
